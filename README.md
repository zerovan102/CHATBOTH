# 🤖 CHATBOTH (Modern AI Chatbot Dashboard)
<img width="1919" height="945" alt="image" src="https://github.com/user-attachments/assets/bd598489-4b1e-4dd1-a0c3-52e806f58fff" />
CHATBOTH adalah proyek web aplikasi dashboard chatbot interaktif dan modern yang dibangun menggunakan framework **Laravel**. Aplikasi ini dilengkapi dengan antarmuka bertema terang (light-themed), desain *glassmorphism* modern, serta berbagai fitur interaktif canggih yang diintegrasikan langsung dengan **Groq API (Llama 3.3)** untuk memberikan respon percakapan yang sangat cepat dan stabil.

---

## ✨ Fitur Utama

- **Integrasi Groq API (Llama 3.3):** Chatbot AI super cepat dengan respons cerdas.
- **Persistent Medium-Term Memory:** Obrolan Anda akan tersimpan secara otomatis menggunakan LocalStorage (hingga 90 hari). Anda tidak akan kehilangan riwayat percakapan.
- **Sidebar Navigasi Responsif:** Navigasi antarmuka yang mulus di berbagai ukuran layar.
- **Interactive Stat Cards:** Menampilkan statistik atau informasi profil secara dinamis.
- **Dynamic Notifications:** Sistem notifikasi *real-time* untuk aktivitas penting.
- **Manajemen Profil & Bintang (Starred Messages):** Personalisasi profil serta kemudahan menyimpan/menandai pesan favorit.
- **Pencarian Real-Time:** Filter dan temukan riwayat pada obrolan (chat) dengan cepat.
- **Pengaturan Lengkap:** Kustomisasi tema, pemilihan model AI, dan kontrol tampilan lainnya sesuai preferensi pengguna.

## 🛠️ Teknologi & Stack

* **Backend:** [Laravel](https://laravel.com/) (PHP)
* **Frontend:** HTML5, CSS3 (Modern Glassmorphism Design), Vanilla JavaScript
* **AI Engine:** [Groq API](https://groq.com/) (menggunakan model Llama 3.3)
* **Penyimpanan Lokal:** Browser LocalStorage
